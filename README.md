# Vocatio Ventures Starter Website

This is a simple static starter site for **Vocatio Ventures**.

## Files
- `index.html` — homepage
- `styles.css` — styling
- `PROMPT-FOR-CODEX-OR-CLAUDE.md` — copy/paste prompt to expand the site

## Quick Use
1. Open the folder in Codex or Claude Code.
2. Give it the prompt in `PROMPT-FOR-CODEX-OR-CLAUDE.md`.
3. Ask it to:
   - keep the brand elegant, warm, and conviction-driven
   - preserve the core message of calling, formation, and execution
   - expand the site into a fuller brand presence if desired
